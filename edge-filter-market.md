<p><ul><li>Long Pass Wave</li><li>Short Pass Wave</li></ul></p>
<p><p>Edge Filter Market Types can be categorized as Long Pass Wave and Short Pass Wave filters. </p><p>Long Pass Wave filters allow longer wavelengths to pass through while blocking shorter wavelengths. They find applications in areas where only specific wavelengths are required.</p><p>Short Pass Wave filters, on the other hand, allow shorter wavelengths to pass through while blocking longer wavelengths. They are used in situations where only certain wavelengths need to be transmitted. </p><p>Both filter types play a crucial role in controlling light transmission based on wavelength.</p></p>
